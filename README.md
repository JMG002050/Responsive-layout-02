## What I Learned

While building this project, I practiced:

* Starting the layout with mobile first approach, using percentages and max-width to control the elements
* Positioning elements relative to one another
* Structuring HTML for maintainability and readability

## Challenges

- One of the biggest challenges was about styling the parent element correctly to make sure the child elements followed the required pattern
- Another challenge was with CSS and styling it correctly, I styled the child elements rather than the parent element
